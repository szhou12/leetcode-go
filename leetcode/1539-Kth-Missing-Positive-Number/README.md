# [1539. Kth Missing Positive Number](https://leetcode.com/problems/kth-missing-positive-number/description/)


## Solution idea

### Binary Search 猜答案

* 对当前一个猜测的答案 `mid`, 
    * 小于它的自然数总数是 `mid - 1` (`[1, mid-1]`).
