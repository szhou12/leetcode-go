# [3. Longest Substring Without Repeating Characters](https://leetcode.com/problems/longest-substring-without-repeating-characters/description/)

## Solution idea

### Sliding Window
* **吃了吐**
    1. 什么时候应该扩大窗口？(吃)
    2. 什么时候应该缩小窗口？(吐)
    3. 什么时候应该更新答案？

Time complexity = $O(n)$
