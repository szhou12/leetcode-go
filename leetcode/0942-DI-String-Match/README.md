# [942. DI String Match](https://leetcode.com/problems/di-string-match/)

## Solution idea

**Greedy Algorithm**: Extract `IDD...D` pattern and append in decreasing order

**Note**: `permutation` means no single number can be used repetitively

Time complexity = $O(n)$
