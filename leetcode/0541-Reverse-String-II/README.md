# [541. Reverse String II](https://leetcode.com/problems/reverse-string-ii/)

## Solution idea

### 模拟题 - 按照题意解

**难点**: 题目要求主要是 每一个 `2*k` 区间反转前k个, 所以要每`2*k `一跳， 要找的也就是每 `2*k` 区间的起点

Time complexity = $O(n)$
