# [189. Rotate Array](https://leetcode.com/problems/rotate-array/)

## Solution idea

### 翻转字符串

1. 整体翻转

2. 局部翻转：前k个为整体进行翻转

3. 局部翻转：后面除前k个以外所有元素为一个整体进行翻转

Time complexity = $O(n)$

Space complexity = $O(1)$
