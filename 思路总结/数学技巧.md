# 常用数学技巧

## 十进制
### 从integer n取digits
* `n % 10` 得到末位
* `n / 10` 砍掉末位

### 把digits组合成integer n
```
num := 0
for each digit {
    num = (num * 10) + digit
}
```



## 因子/Divisor
### GCD (Greatest Common Divisor)
* Euclidean Algorithm (辗转相除法)
```go
func gcd(a, b int) int {
    if b == 0 {
        return a
    }
    return gcd(b, a % b)
}
```

### LCM (Least Common Multiple)
`lcm(a, b) = a * b / gcd(a, b)`
