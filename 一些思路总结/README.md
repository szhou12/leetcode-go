# Handbook

## Emojis

| ico | meaning |
| :-: | - |
| :star: | 类型题的总结性语言 |
| :red_circle: | 困难 |
| :green_circle: | 简单, 可直接出答案 |
| :lock: | 加锁题 |
| :secret: | 解题使用了值得记住的技巧 |

## Readme content structure

每道题的Readme结构统一如下:

* Solution idea
    * [solution type]
        * 要点总结
        * 物理意义
        * 代码整体结构总结
        * Time complexity & Space complexity
* Resource
